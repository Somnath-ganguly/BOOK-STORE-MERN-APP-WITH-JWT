# 📚 BookStore MERN App with JWT

A full-stack BookStore Management System built using the MERN Stack (MongoDB, Express.js, React, Node.js) with JWT Authentication and bcrypt password hashing.

## 🚀 Features

* 🔐 Secure user authentication with **JWT (JSON Web Token)**.
* 🔒 Passwords are encrypted using **bcrypt** before storing them in the database.
* 📚 Perform complete **CRUD operations** (Create, Read, Update, Delete) for book management.
* ⚡ RESTful APIs built with **Node.js** and **Express.js** for efficient client-server communication.
* 💾 Store and manage data using **MongoDB** with **Mongoose**.
* 🎨 Responsive and user-friendly interface built with **React.js**.
* 🔑 Protected routes ensure that only authenticated users can access authorized resources.


## 🛠️ Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt
